# proyecto-final-base-de-datos-gestor-de-equipos-y-reportes
El programa gestor de equipo y reportes, es un sistema diseñado para ayudar en la operatividad dentro del ambito hospitalario, este ayuda con la organización y gestión de los equipos, manteniendo una coherencia y un flujo constante y seguro sobre la interaccion entre el tecnico o biomedico encargado de llevar el contro y organizaón de los equipos, de esta forma el Técnico tiene total control de que equipos tiene en inventario, que equipos tienen daños, que equipos necesitan mantenimiento y que equipos estan fuera de servicio, de esta manera el Técnico puede realizar reportes nuevos sobre los equipos para mantener la situación actualizada y poder dar seguimiento al proceso de reparacion del equipo en cuestion, del mismo modo el sistema provee de una opcion que permite descargar el reporte o los reportes de mantenimiento de los equipos, de esta manera el Técnico puede llevar consigo los reportes actuales en fisico para no tener la necesidad de cargar con una PC, en general el sistema permite fomentar la organización y del mismo modo mejorar la operatividad intrahospitalaria y permanecer en control en todo momento.
